## Overview

The **Field Health Services Information System (FHSIS) Manual of Operations 2018** is the authoritative DOH reference for the Philippines' nationwide, facility-based public health data collection system. It governs how Barangay Health Stations (BHS), Rural Health Units (RHU), and Main Health Centers (MHC) record, consolidate, and report health service coverage data upward through the municipal/city, provincial/CHO, regional (CHD), and national (DOH-EB) levels. The MOP defines every indicator tracked by the system — its exact formula, eligible population denominator, target, data source, and reporting frequency — across six program clusters: Family Health Services (FP, maternal, child, oral health), Infectious Disease Prevention, NCD Prevention, Environmental Health, plus annual morbidity/mortality/demographic data. It also specifies the complete set of recording and reporting forms (ITR, TCL, Summary Table, MCT/ACT, M1, M2, Q1, A1), the flow of reports at every administrative tier, the roles and responsibilities of each health cadre, and guidelines for data validation, analysis, dissemination, and monitoring FHSIS implementation timeliness and completeness.

## [Chapter 9.1 — Data Validation]
- **File:** `fhsis_mop/09_1_data_validation.md`
- **Pages:** PDF 461–464 (doc pages 447–450)
- **Summary:** Three data validation schemes with worked examples using actual FHSIS table values: (1) within-dataset validation — checking that sub-totals by sex/weight/birth attendant sum to the correct grand total; (2) proportion/percentage checks — flagging values >100% of eligible population (e.g., Vitamin A given > estimated children); (3) cross-dataset validation — confirming that the same denominator (e.g., estimated pregnant women) is used consistently across multiple indicators. Also covers validation of empty cells (distinguish zero from missing data from N/A). Critical reference for building any data quality check logic.
- **Tags:** data validation, data quality, cross-check rules, sum totals, percentage over 100, empty cells zero vs no-data, cross-dataset consistency, live birth totals, immunization coverage check, facility delivery vs SBA check, IMR vs UFMR check, denominator consistency, pregnant women denominator, FHSIS data quality rules
